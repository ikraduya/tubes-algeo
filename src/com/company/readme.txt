====Cara Penggunaan Aplikasi====

Aplikasi ini dapat menyelesaikan 2 permasalahan yaitu
Sistem Persamaan Linear dan Interpolasi

1. Pertama-tama akan keluar UI Menu sebagai berikut

    MENU
    1. Sistem Persamaan Linear
    2. Interpolasi Problem
    3. Selesai
    Pilihan :

Pilihlah pilihan menu yang diinginkan.

2. Setelah itu akan muncul menu sebagai berikut

    Metode Input
    1. File txt eksternal
    2. Matriks masukkan
    Pilihan :

Pilihlah metode input yang diinginkan.

3. Jika memilih menu 1, maka anda harus menulis kan nama file eksternal yang ingin dibaca

4. Jika memilih menu 2, maka anda harus menentukan jumlah baris dan kolom matriks
lalu mengisi matriks secara manual

5. Setelah itu akan muncul menu sebagai berikut

    Metode Penyelesaian
    1. Eleminasi Gauss
    2. Eleminasi Gauss-Jordan
    Pilihan :

Pilihlah metode penyelesaian yang diinginkan

6. Setelah itu akan keluar solusi SPL atau Interpolasi ke layar anda

7. Anda bisa menyimpan solusi tersebut ke dalam file eksternal sesuai keinginan anda
